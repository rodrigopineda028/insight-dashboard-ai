# Archivos de Ejemplo

Este directorio contiene archivos de ejemplo para probar la aplicación.

## ventas_ejemplo.csv

Dataset de ventas con 30 registros que incluye:

- **Categorías**: Electrónica, Hogar, Ropa, Alimentos
- **Regiones**: Norte, Sur, Este, Oeste
- **Métricas**: Ventas (monto), Unidades vendidas
- **Temporal**: Fechas de enero-febrero 2024

Ideal para probar:

- Gráficos de barras por región
- Análisis de ventas por categoría
- Series temporales
- Comparaciones de productos
